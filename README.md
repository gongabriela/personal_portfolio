# Personal Portfolio Website

Welcome to my first web project! This work was proposed for the **Web Fundamentals** curricular unit of the **UPskill ServiceNow 25/26** program, where we were asked to create a web portfolio. For my portfolio, I sought to learn and implement HTML and CSS concepts, as well as some light JavaScript to implement dropdown menu functionalities.

## About the Project
This repository aims to apply and consolidate concepts learned in the module. The project is dedicated to creating a personal website that serves as a portfolio and curriculum vitae.

## Technologies Used
- **HTML5**: Semantic structure (Header, Sections, Footer).
- **CSS3**: Styling using Flexbox for layout, and `rem`/`px` units for responsiveness.
- **JavaScript**: Basic interactivity for dropdown menus.

## Structure
Based on the project requirements, the website includes:
- **Header**: Navigation and branding.
- **Project Cards**: Display of portfolio items.
- **Filter Dropdown**: To sort or filter projects.
- **"All Projects" Block**: A section listing all works.
- **About Me Page**: Personal introduction.
- **Footer**: Contact info and links.

## How to Access
1. Clone the repository.
2. Open the `index.html` file in your preferred web browser.

The Wiki section of this repository is available, with detailed pages where I explain the concepts learned and implemented for this project. Feel free to explore the Wiki and my line of reasoning behind the code!

# Website de Portfólio Pessoal

Olá e muito bem-vindos ao meu primeiro projeto web! Foi proposto um trabalho para a unidade curricular **Fundamentos Web** do programa **UPskill ServiceNow 25/26**, onde nos é pedido para criar um portfólio web. Para o meu portfólio busquei aprender e implementar alguns conceitos de HTML e CSS, bem como um leve JavaScript para implementar algumas funcionalidades de drop down menu.

## Sobre o Projeto
Este repositório destina-se a aplicar e consolidar os conceitos aprendidos no módulo. O projeto dedica-se à criação de um website pessoal que sirva como portfólio e curriculum vitae.

## Tecnologias Utilizadas
- **HTML5**: Estrutura semântica (Header, Sections, Footer).
- **CSS3**: Estilização usando Flexbox para layout, e unidades `rem`/`px`.
- **JavaScript**: Interatividade básica para menus drop down.

## Estrutura
Baseado nos requisitos do projeto, o website inclui:
- **Header**: Navegação e branding.
- **Card de Projetos**: Exibição dos itens do portfólio.
- **Drop Down Menu de Filtros**: Para ordenar ou filtrar projetos.
- **Bloco de "All Projects"**: Uma secção listando todos os trabalhos.
- **Página do "Sobre Mim"**: Introdução pessoal.
- **Footer**: Informações de contato e links.

## Como Acessar

2. Abra o arquivo `index.html` no seu navegador de preferência.
