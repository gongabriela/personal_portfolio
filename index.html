Cub3D

Um mecanismo de raycasting

O Cub3D é um motor gráfico simples que utiliza a técnica de raycasting para renderizar um ambiente 3D a partir de uma perspectiva em primeira pessoa. Inspirado no clássico jogo Wolfenstein 3D, este projeto foi desenvolvido em C, utilizando a biblioteca gráfica MiniLibX.

Conceitos Utilizados

  • Raycasting
  • Manipulação de Imagens em C
  • Geometria Computacional
  • Leitura e Parsing de Ficheiros
